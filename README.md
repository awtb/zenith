# Zenith

**Zenith** is a minimal self-hosted tunneling tool for exposing local development services to the internet built with Go.

The project is inspired by ngrok, but its initial goal is not to build a SaaS platform. Zenith is built for personal
use and understanding networking fundamentals through a real system.

Status

Early design stage.

No stable API, CLI, or protocol yet.
